# [cassiandloganaregettingmarried.com](https://cassiandloganaregettingmarried.com)

Wedding website for Cassi and Logan's wedding at Gold Hill Inn, Gold Hill, CO — July 22, 2023.

Handcrafted HTML and CSS. No frameworks, no build steps, no JavaScript* — just plain ol' humble ol' markup.

Built on [Skeleton](http://getskeleton.com/) for a lightweight responsive grid and [Raleway](https://fonts.google.com/specimen/Raleway) for type.

\* _The RSVP page had a small inline script for form toggling. We weren't monsters, just minimalists._
